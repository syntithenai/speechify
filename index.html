Springy Map Demo

This demo builds on the speech recognition available in Chrome and Android browsers to create a speech based mindmap editor.

This demo was intended to show the possibilities for speech recognition in virtual reality computing environments.

The editor makes available a range of speech commands to support the editing of a mindmap so you can say things like -

Note Management

  • Add note [name of new note]
  • Delete note [name of note]
  • Select note [name of note]
  • Rename note [name of note] to [new name of note]
  • Move selected note to [name of new note]

Note Editing

You can edit the text inside a note by saying
  • Edit note [name of note]
  • In editing mode everything you say is dictated after the current text selection except voice commands like
    • Select [text to select]
    • Replace selected with [text to replace]
    • Delete that
    • Comma
    • Fullstop
    • Finish editing

Other commands include

  • What can I say
  • Save/List/Load/Delete/Clear map
  • Render as text/virtual reality/mindmap
  • Wake up/Go to sleep/Start Listening/Pause Listening/Stop Listening

Microphone Status

When the microphone is green, speech recogntition is ready and listening.

After a period of silence, listening will pause automatically and the microphone will turn orange. Listening can be restarted by saying "Start Listening".

After a long period of silence, listening will stop automatically and the microphone will turn red. Listening can be restarted by clicking on the microphone.

Documentation

Detailed documentation for the speech grammar parsing and examples showing how to integrate grammar based speech recognition as well as the GPL open source code is available at github https://github.com/syntithenai/speechify

Example usage: 
var speechify = $("body").speechify({
        'grammarTree' : [['do stuff'],function() {console.log('busy doing');}]]
});
Or for a multi stage voice command 
var speechify = $("body").speechify({
[['save file [$file]'], function(parameters) {
    var value = parameters[1]['$file'];
    speechify.requireVariable('$title','What is the title of your document ?',parameters,
        function(value) {
        // success -> have a value for $title
        speechify.confirm('Do you really want to save as '+value+' ? Yes or No',function() {
            jQuery.fn.speechify.notify('Saved file as ' + value);
        });
        },
        function() {
        // fail     
            jQuery.fn.speechify.notify('Cancelled save.);
        }
    ]); 
}});
